Веб інсталятор прошивки NerdMiner

Рекомендовано для: просунутих користувачів.

  1. Підключіть пристрій до комп'ютера через USB. Натисніть кнопку "Install" нижче, виберіть потрібний порт та завантажте прошивку на пристрій.
  2. Після першого запуску пристрій NerdMiner сам створить Wi-Fi мережу з іменем "NerdMinerAP" без паролю або паролем "MineYourCoins".
  3. Підключіться до цієї мережі зі свого пристрою.
  4. Вас автоматично переправить на сторінку пристрою. Якщо ні, відкрийте браузер та перейдіть за адресою: http://192.168.4.1.
  5. Заповніть форму налаштувань, ввівши SSID вашої Wi-Fi мережі, пароль, вкажіть адресу біткойн-гаманця, опціонально пулл та порт.
  6. Натисніть "Save". Пристрій перезавантажиться та спробує підключитися до вказаної Wi-Fi мережі.

Install Firmware on nerdminer

Якщо пристрій не виявлено:

  1. Переконайтеся, що пристрій підключений до комп'ютера, на якому запущений цей веб-сайт.
  2. Перевірте світлодіод на пристрої (якщо він є) — він має світитися.
  3. Переконайтеся, що використовується USB-кабель для передачі даних, а не лише для живлення.
  4. Переконайтеся, що встановлені правильні драйвери:
    • Драйвери для CH340, CH341: Завантажити

Увага: Програмне забезпечення надається як є, без будь-яких гарантій. Користувач несе повну відповідальність за наслідки його використання, включаючи можливі пошкодження обладнання або втрату даних. Адміністрація не відшкодовує жодних збитків.

Щоб придбати прошитий пристрій, обговорити прошивку або отримати підтримку, скористайтеся цим посиланням: Обговорення прошивки.
